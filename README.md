### ЁЯУШ Overview
**Project Madurai** is an open, volunteer-driven initiative that digitizes and publishes ancient Tamil literary works in electronic form (ETEXTS).  
This repository provides **structured book details** from Project Madurai releases in **Google Sheet** formats for research, teaching, and digital preservation.

---

### ЁЯЧВя╕П Contents
The repository contains:
- ЁЯУД **Book Metadata:** Title, author, publication year, edition, and source.
- ЁЯЧГя╕П **Categorization:** Literary type (poetry, grammar, commentary, etc.).
- ЁЯФд **Language Encoding:** Tamil Unicode (UTF-8).
- ЁЯУС **Download Formats:** `.csv`, `.xlsx`, `.pdf`, and `.txt` versions.
- ЁЯМР **Reference Links:** URLs to Project Madurai text pages.

---

### ЁЯТб Purpose
This collection helps:
- Scholars in **Tamil literature and linguistics**.
- Researchers in **Digital Humanities** and **Tamil NLP**.
- Developers working on **Tamil language datasets**.
- Educators preserving classical Tamil texts for **pedagogical use**.

| Field Name | Description |
